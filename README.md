# Traffic Light Simulation

A browser-based traffic light simulation using HTML, CSS, and JavaScript.

## Files

- `index.html` — main simulation page
- `style.css` — styling for the dashboard and canvas
- `script.js` — simulation logic, asset loading, and animation loop
- `bg/`, `signal/`, `left/`, `right/`, `up/`, `down/` — image assets used by the simulation

## Run locally

1. Open `index.html` in your browser.
2. Or host with a simple local server, for example:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser.
